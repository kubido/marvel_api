## HOW TO RUN

### RUN SERVER

- run `npm install`
- rename `.env.example` to `.env` and change the value
- run `redis-server`, if you don't have redis. you need to [install](https://redis.io/topics/quickstart) it first
- run `npm start`

### RUN TEST

- go to root app folder
- run `npm install`
- run `npx jest`
